
cirru/parser
----

Cirru Parser in Clojure.

### Usage

```clojure
[cirru/parser "0.0.1"]
```

```clojure
(ns your-project.core
  [:require [cirru.parser.core :as cirru]])
(cirru/pare "code" "filename")
```
